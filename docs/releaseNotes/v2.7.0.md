## Mansur-Rig 2.7.0
Released 21 Oct 2025

### General
<ul>
	<li><b>Maya 2026 Support for Windows</b>- Due to increasing complications and hardware requirements, Mansur-Rig 2.7 now supports Maya 2026 for Windows platform only. Linux support <b>IS NOT</b> dropped, only delayed.</li>
	<li><b>PyMel library - upgraded to support maya 2026.</li>
</ul>

### Bug Fixes
- Split localClump flip attributes to translate & rotate
- Lips B module- node accumulation bug- Fixed
- Added flip Right option to Splay module
- Maya 2025 DynUI is not working- Fixed
- freeControl- "position mode" is taken into account only in "sameMeshEffector" mode
- Upgrade "FindNextIncrement" function to be able to increment Alpha as well, when the 999 ID limit is passed
